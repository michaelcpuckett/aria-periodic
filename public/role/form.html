• Landmark

The form role

Description

A landmark region that contains a collection of items and objects that, as a whole, combine to create a form.

Allowed Content

Flow content, but with no form element descendants.

Content Category
  • Flow Content
  • Palpable Content
HTML Elements with Implicit ARIA Role
  • <form>
Allowed HTML Elements
  • <form> (role attribute unnecessary)
  • <div>, <span>, <p>, other elements that can receive any role
