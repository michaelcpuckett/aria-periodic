Live Region

The marquee role

Description

A type of live region where non-essential information changes frequently.

Allowed Content

Flow content, but with no main element descendants.

Content Category
  • Flow Content
HTML Elements with Implicit ARIA Role
  • (None)
Allowed HTML Elements
  • <section>
  • <div>, <span>, <p>, other elements that can receive any role
Specification Links
