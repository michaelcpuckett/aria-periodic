The columnheader* role

Description

A cell containing header information for a column.

Abstract Role

Structure*, Widget*

*May be interactive or non-interactive depending on the context: Interactive when inside a grid or treegrid.

Allowed Content

Flow content but with no main, header, or footer element descendants.

Content Category
    Required Context Roles
    • row
    HTML Elements with Implicit ARIA Role
    • <th[scope="col"]>
    Allowed HTML Elements
    • <th>
    • <th[scope="col"]> (role attribute unnecessary)
    • <div>, <span>, <p>, other elements that can receive any role
